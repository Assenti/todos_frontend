# Personal Planner ver. 2.0 web app frontend part on Vue.js (written on TypeScript)

> Used Stack: TypeScript, Vue.js 2, Vuetify 2

### App features
- Creating personal todos list
- Todo updating and deleting
- Toggle completion and importance of a todo
- Sorting todos by date, completion or importance filters
- Watching todos in the calendar
- Sending todos list via email
- Fetching personal todos list or group\project todos list
- Creating group or project and adding todos there
- Group or project updating and deleting 
- Selecting performer of a todo
- Password restoring & changing in the account block
- Updating an account data
- Sending invitation to friends via email
